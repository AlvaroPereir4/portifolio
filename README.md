# Portfólio Pessoal

Este é um site de portfólio pessoal para exibir informações, habilidades e projetos.

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

## Funcionalidades

* Seção "Sobre Mim" com links para redes sociais e currículo.
* Seção de habilidades com ícones representando as tecnologias.
* Carregamento dinâmico de projetos e links a partir de arquivos JSON.
* Design responsivo para se adaptar a telas de desktop e dispositivos móveis.

## Como Executar

Basta abrir o arquivo `index.html` em qualquer navegador web. Os dados dos projetos e links pessoais são carregados a partir dos arquivos na pasta `/data`.
